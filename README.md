# How to build
Run **Basic_2D_OpenGL_Game.sln** in Visual Studio 2015. Open Project->Properties from the menu and then set the following settings:

**Configuration:** Debug

**Platform:** Win32

**General->Platform Toolset:** Visual Studio 2015 (v140)

**Apply & OK & Build**
